# KGoT Image Setup

## Steps to Run (Docker)

In order to build and run the KGoT Docker image, please navigate to the `kgot` directory and execute:

```bash
docker-compose up 
```

## Steps to Run (Sarus)

In order to build and run the KGoT Docker image, please navigate to the `kgot` directory and execute:

```bash
chmod +x sarus_launcher.sh     # If needed
./sarus_launcher.sh
```
